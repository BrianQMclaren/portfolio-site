User Authentication Form

  • Role - Front End Developer
  • Registration & Login Form
  • July 2017

Project History

As a Front end developer this was my first attempt at building a full stack javascript project. My goal was to create a register & login form that validates, authenticates and stores user input into a database. I was undecided as to make this a full app hence just barebones landing page. I may revist this project in the future.

Technologies Used

Express, Node, EJS, MongoDB, ES6

User Stories

  • Authentication system validates users when logging in or creating a new account.
  • Separate express.js route for Login Form.
  • Submits to mongoDB database and stores infomation.
  • Hashes and salts passwords to protect user profile security via bcrypt technology.
  • Redirects users to error page when login infomation doesn't match.

Project Analysis

  1. I created this project with javascript technologies such as Node.js, Express framework, and Ejs template language. For a database I chose to use MongoDB to store user entries while also creating a mongoose schema to structure my document shapes within a collection.
  2. For my web app to handle state I leverged express sessions to provide each user with a unique session with each login. Which means based on what action a user took on page 1, we will show a different page 2.
  3. Finally I implemented security measures to ensure user info is stored safely. Choosing Bcrypt to hash and salt passwords once stored in the database created a scrambled string from the original password. Therefore hackers cannot unhack the password which ensures users infomation is secure.
